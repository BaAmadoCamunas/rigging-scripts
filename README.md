# Maya Rigging Tools

A collection of Python tools developed for Autodesk Maya to automate rigging workflows, improve production efficiency and simplify repetitive tasks.

These scripts were created during my technical art and rigging training, focusing on clean code, usability and practical production workflows.

---

## Repository Overview

This repository contains standalone rigging utilities designed to solve common production problems.

Current tools include:

| Tool | Description |
|------|-------------|
| Color Panel Tool | Professional UI for viewport color management and Arnold curve render settings. |
| Offset Parent Matrix Freeze | Freeze transforms using `offsetParentMatrix` while preserving hierarchy and world transforms. |

More tools will be added over time as the library grows.
